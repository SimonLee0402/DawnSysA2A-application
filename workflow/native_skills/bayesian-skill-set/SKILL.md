# Bayesian Skill Set

This is a Dawn native builtin skill.

Purpose:
- perform uncertainty-aware planning
- choose the safest next step under incomplete evidence
- map operator intent into the four Dawn safety levels

Primary local surfaces:
- `#chat`
- `#observe`
- `#assist`
- `#autopilot`
- `/app` Command Studio

Notes:
- this is a native Dawn workflow skill, not a Wasm artifact
- it is always available on the local system once Dawn is installed
